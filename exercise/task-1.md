# Tugas: Mengenal Struktur Framework Laravel

- Tugas pertemuan pengganti 4
- Mata Kuliah Pemrograman Web Berbasis Framework

---

## Instruksi Tugas

Mahasiswa diminta untuk melakukan **eksplorasi terhadap struktur project Laravel** yang sudah dibuat pada pertemuan sebelumnya.

---

## Bagian 1

### Mengamati Struktur Folder Laravel

Perhatikan struktur folder pada project Laravel Anda.

Jelaskan fungsi dari folder berikut:

| Folder    | Penjelasan yang Harus Ditulis Mahasiswa |
| --------- | --------------------------------------- |
| app       |                                         |
| bootstrap |                                         |
| config    |                                         |
| database  |                                         |
| public    |                                         |
| resources |                                         |
| routes    |                                         |
| storage   |                                         |
| tests     |                                         |
| vendor    |                                         |

Mahasiswa diminta menjelaskan **fungsi utama setiap folder** minimal **3–4 kalimat**.

---

## Bagian 2

### Mengenal Folder Penting dalam Folder `app`

Buka folder:

```
app/
```

Jelaskan fungsi dari:

| Folder    | Fungsi |
| --------- | ------ |
| Http      |        |
| Models    |        |
| Providers |        |

Tambahkan juga penjelasan:

**Apa fungsi Controller dalam Laravel?**

---

## Bagian 3

### Mengenal Folder `routes`

Buka folder:

```
routes/
```

Jelaskan fungsi file berikut:

| File         | Fungsi |
| ------------ | ------ |
| web.php      |        |
| console.php  |        |
| channels.php |        |
| api.php      |        |

---

## Bagian 4

### Mengenal Folder `resources`

Buka folder:

```
resources/
```

Jelaskan fungsi dari:

| Folder | Fungsi |
| ------ | ------ |
| views  |        |
| css    |        |
| js     |        |

Tambahkan juga:

**Apa itu Blade Template dalam Laravel?**

---

## Bagian 5

### Analisis Arsitektur Laravel

Jawab pertanyaan berikut:

1. Laravel menggunakan **arsitektur apa dalam pengembangan web?**
2. Apa perbedaan **framework dengan native PHP**?
3. Mengapa Laravel banyak digunakan dalam pengembangan web modern?

Minimal **1 paragraf tiap pertanyaan**.

---

## Output yang Dikumpulkan

Mahasiswa mengumpulkan **laporan PDF** berisi:

1. Screenshot struktur project Laravel
2. Penjelasan setiap folder
3. Jawaban pertanyaan analisis

Minimal **3–4 halaman**.
